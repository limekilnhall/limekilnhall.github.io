Twelve Basic Moves

The basic moves are intended to help the student learn the key techniques and stances. The stances and cutting moves learned here are used throughout all the more advanced techniques. These moves are repeated during every class to help the student refine the technique as well as to build up flexibility, stamina and power.

Because of the repetition involved, it is useful to learn how to count in Korean.

Counting in Korean

  1. Hana
  2. Dul
  3. Set
  4. Net
  5. Dasut
  6. Yusut
  7. Ilgob
  8. Yudul
  9. Ahop
  10. Yul
